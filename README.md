# Tic-Tac-Toe (Python)

## About this project
I made this Tic-Tac-Toe game using Python and Tkinter while learning how GUI works. I wanted to build something simple that I can actually see and play, instead of just console programs.

In the beginning, I found it a bit confusing to handle button clicks and game logic together, but after trying a few times and fixing mistakes, I understood it better.

## What it does
- Two players can play (X and O)
- Shows whose turn it is
- Detects the winner
- Highlights the winning boxes

## How to run
Just run the file:
python main.py

## What I learned
- Basics of Tkinter
- How to handle user input
- Writing simple game logic

## Future plans
I want to add a restart button and maybe make a mode where we can play against the computer.
